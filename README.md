# Playlist Generator

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode with webpack-dev-server
$ npm run watch

# without webpack-dev-server
$ npm install -g live-server

# build development mode
$ npm run dev

# build production mode
$ npm run build

# run
$ live-server dist --verbose
```

## Test

```bash
$ npm test
```
